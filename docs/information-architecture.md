# Portfolio OS 信息架构

## 定位

这是一个结构化能力展示系统，不是传统简历网站。核心目标是让招聘方在 3-5 分钟内理解：

- 你能处理什么类型的问题
- 你用什么方法处理
- 你有什么可验证证据
- 你如何把 AI/Vibe Coding 转化为真实产出

## 一级导航

1. Home
   - 个人定位
   - 能力关键词
   - 核心指标
   - 能力增长趋势图

2. Projects
   - 使用 Case Study 结构
   - 每个项目包含 Problem / Action / Result
   - 标签化展示方法与能力

3. Insight Dashboard
   - 将履历数字转为数据产品视角
   - 展示指标、方法、状态
   - 强化数据分析与业务解释能力

4. AI Workflow
   - 展示 Prompt 到 Demo 的完整链路
   - 突出 Vibe Coding、AI 协同开发、自动化工作流
   - 后续可扩展为真实项目截图、代码仓库、在线 demo

5. Contact
   - 求职方向
   - 邮箱
   - 电话

## Case Study 模板

每个项目固定使用：

- Problem：业务问题、约束、痛点
- Action：你采取的动作、工具、流程
- Result：交付结果、指标、可复用经验
- Tags：能力标签，用于连接岗位 JD
