# Zhang Jiahao Portfolio OS

A structured personal portfolio system for showcasing data analysis, product thinking, AI workflow, and Vibe Coding capability.

This is not a simple resume page. It is designed as a capability dashboard with case studies, metrics, workflow evidence, and contact entry points.

## Information Architecture

- `Home`: positioning, capability metrics, and a data-style overview console
- `Projects`: case studies organized by `Problem / Action / Result`
- `Insight Dashboard`: portfolio metrics translated into data product language
- `AI Workflow`: prompt-to-prototype workflow and Vibe Coding learning path
- `Contact`: email, phone, and target role direction

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- ECharts
- lucide-react
- motion
- clsx
- pnpm

## Run Locally

```bash
pnpm install
pnpm dev
```

Quality checks:

```bash
pnpm lint
pnpm build
```

## Notes

Local resume source materials and generated diagnostics are intentionally ignored by Git. Public-facing project structure is documented in `docs/information-architecture.md`.
